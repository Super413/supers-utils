# supers-utils
A wonderful storage space of scripts

## What is this?
This is my personal little stash of code utils I use or have used for various things. Most of the scripts are in python 3, but other languages are used occasionally. <br><br> All of these are either coded fully by me, or vibe coded <br> Generally, games are coded with Gemini pro, as HTML files with JS and CSS embedded. <br><br>Most are in a playable or semi-playable state, expect bugs.<br>If a game is interesting, I tend to fork it and develop it further on my own or with codex.

# How to use
Access a helpful little portal [here](https://super413.github.io/supers-utils/)
<br>If forking to add your own scripts, run ```generate.js``` to create a ```generated.json``` file, this acts as the search index it will be based off.
